# CESMD - Center for Engineering Strong Motion Data

## Rol

Centro de datos strong motion para acelerogramas, metadatos de eventos, estaciones y registros.

## Documentación oficial

- https://strongmotioncenter.org/
- https://strongmotioncenter.org/wserv/
- https://www.strongmotioncenter.org/wserv/records/documentation/
- https://strongmotioncenter.org/wserv/events/builder/

## Servicios

CESMD Web Services permite consultar:

- earthquake metadata
- station metadata
- record metadata
- record datasets

## Formatos

Para metadatos de records se documentan formatos:

- json
- xml

## Campos/filtros observados en documentación records

- evname
- faulttype
- format
- groupby

## Uso en EarthquakeAnalysis

- Dataset strong motion complementario a ESM.
- Acelerogramas para validación de modelos de daño.
- Comparación PGA/PGV/SA contra distancia y tipo de falla.
